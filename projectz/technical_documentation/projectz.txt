== ((Analisi funzionale)) ==
La piattaforma è suddivisa in due parti:

-----------------------------------------------------
	* Parte pubblica (da qui in avanti definita come "passerella")
	* Parte privata (da qui in avanti definita come "profilo privato" )
-----------------------------------------------------

Qui di seguito verranno definite le funzionalità messe a disposizione dalla passerella e dal profilo


=== Passerella ===
Le funzionalità rilevate sono:

-----------------------------------------------------
	* Iscrizione alla piattaforma
	* Visualizzazione passerella
	* Visualizzazione book partecipante
	* Votazione partecipante
	* Aggiunta di un partecipante all'elenco preferiti
-----------------------------------------------------

=== Profilo privato ===
Le funzionalità rilevate sono:

-----------------------------------------------------
	* Modifica dei dati del profilo
	* Aggiunta/rimozione foto per la passerella
	* Visualizzazione statistiche (visite, punti, posizione in classifica, ...)
	* Eliminazione del profilo
-----------------------------------------------------

== ((Analisi relazionale)) ==
Definizione della struttura del database relazionale.
=== Entita' rilevate ===
Le entità rilevate sono le seguenti:

-----------------------------------------------------
	* Contests
	* Participants
	* Contest_Stats
	* Votes
	* Metrics
-----------------------------------------------------

=== Attributi delle entita' ===
Gli attributi rilevati per ogni entità:

==== Contests ====

-----------------------------------------------------
	* id_contest (PK)
	* start_date
	* end_date
	* name
	* description
	* status (0 in attesa di apertura, 1 attivo, 2 chiuso)
-----------------------------------------------------

==== Participants ====

-----------------------------------------------------
	* id_participant (PK)
	* first_name
	* last_name
	* username
	* email
	* password
	* city
	* country
	* gender
	* creation_date
	* update_date
-----------------------------------------------------

==== Contest_Stats ====

-----------------------------------------------------
	* id_contest_stat (PK)
	* id_participant
	* id_contest
	* id_metric
	* points
-----------------------------------------------------

==== Votes ====

-----------------------------------------------------
	* id_vote (PK)
	* id_participant
	* id_contest
	* ip_address
	* date
-----------------------------------------------------

==== Metrics ====

-----------------------------------------------------
	* id_metric (PK)
	* name
	* min_limit
	* max_limit
-----------------------------------------------------
